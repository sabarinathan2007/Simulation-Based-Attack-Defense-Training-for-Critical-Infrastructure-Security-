## 🚀 Quick Start (15 Minutes)

### 1. Run Setup Script
```bash
# Linux/Mac
bash setup.sh

# Windows
setup.bat
```

### 2. Start Services (3 terminals)
```bash
# Terminal 1: MQTT
docker-compose up -d mosquitto

# Terminal 2: Backend
cd backend && python app.py

# Terminal 3: Frontend
cd frontend && npm start
```

### 3. Login & Explore
```
http://localhost:3000
Username: user1
Password: demo
```
